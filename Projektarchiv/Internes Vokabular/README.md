# Internes Vokabular
> Mai 2023

Es gibt bestimmte interne Informationen zu einem Projekt, die nicht öffentlich zugänglich sein sollen, beispielsweise Kommentare zur Erfassung oder zu den Nutzungsrechten. In Madek unterliegen Metadaten dem Permissions-System. Vokabulare können auf Nutzer oder Gruppen eingeschränkt werden.

Wir verwenden ein zugansbeschränktes Vokabular `internal`.

Zu unterscheiden: technische (nicht semantische) Metadaten im Vokabular `settings`.
